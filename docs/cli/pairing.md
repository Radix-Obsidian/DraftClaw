# `draftclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
draftclaw pairing list whatsapp
draftclaw pairing approve whatsapp <code> --notify
```
