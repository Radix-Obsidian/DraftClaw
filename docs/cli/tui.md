# `draftclaw tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/tui)

## Examples

```bash
draftclaw tui
draftclaw tui --url ws://127.0.0.1:18789 --token <token>
draftclaw tui --session main --deliver
```
