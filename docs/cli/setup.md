# `draftclaw setup`

Initialize `~/.draftclaw/draftclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
draftclaw setup
draftclaw setup --workspace ~/.draftclaw/workspace
```

To run the wizard via setup:

```bash
draftclaw setup --wizard
```
